== Description ==

WooCommerce Admin Customer Milestone Notifier will Notify you when your store will have 1,10,100,250,500,1000,10000 customers registration. It will require, WooCommerce and WooCommerce Admin both the plugin.

A few notes about the sections above:

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `customer-milestone-notifier-woocommerce-admin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
